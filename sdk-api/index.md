# NNark SDK API Reference

Auto-generated API documentation for the .NET Ark SDK (**NNark**) libraries bundled with the plugin.

## Libraries

| Library | Description |
|---|---|
| **NArk.Abstractions** | Interfaces and domain types: storage abstractions, wallet interfaces, core domain records (ArkCoin, ArkVtxo, ArkContract, ArkAddress) |
| **NArk.Core** | Wallet implementations, VTXO logic, HD/SingleKey signers, intent generation, batch management |
| **NArk.Swaps** | Boltz submarine and reverse swap client, chain swap MuSig2 signing |
| **NArk.Storage.EfCore** | Entity Framework Core storage implementations for VTXOs, contracts, intents, swaps, and wallets |

> [!NOTE]
> The NNark SDK is maintained in a [separate repository](https://github.com/arkade-os/dotnet-sdk) and included as a Git submodule. This reference documents the version bundled with the current plugin release.
